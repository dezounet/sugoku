# Sugokud

Go daemon serving a multiplayer sudoku game.

## Build from sources

```bash
make all
```

Binary files are generated in `cmd/sugokud` directory.

## Contribute

### Commit changes

Before any commit, install pre-commit hooks:

```bash
pre-commit install
```
